
# Tourism Management System (TMS)

## Technologies Used
- PHP
- MySQL
- HTML, CSS, Bootstrap
- JavaScript
- XAMPP
- VS Code

## How to Run the Project
1. Install XAMPP
2. Copy project folder to htdocs
3. Start Apache & MySQL
4. Import `tms.sql` into phpMyAdmin
5. Run: http://localhost/tms

## Index Panel
- URL: http://localhost/tms/index.php
